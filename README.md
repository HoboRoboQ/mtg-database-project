MTG Card Manager
A simple command-line tool for managing a personal Magic: The Gathering card database.

What is it:
A Python CLI application that allows users to store, search, and manage MTG cards locally using a JSON file.

Features:
- Search cards (English or Japanese name)
- Add new cards
- Delete cards with confirmation
- View sorted card list
- Persistent storage via JSON

Notes:
Built as a learning project to practice:
- File I/O
- JSON data handling
- CLI design
- Git & GitHub workflow
